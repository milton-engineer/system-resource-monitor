# System Resource Monitor

Lightweight Linux monitoring utility focused on real-time system resource inspection.

## Overview

System Resource Monitor is a simple command-line tool designed to display critical Linux system information such as CPU usage, memory consumption and uptime.

The project simulates infrastructure monitoring workflows commonly used by Linux administrators and backend engineers.

## Features

- CPU monitoring
- Memory usage analysis
- System uptime display
- Lightweight terminal interface
- Linux environment diagnostics

## Technologies

- Python
- Linux
- Bash

## Goals

- Practice Linux system monitoring
- Simulate infrastructure tooling
- Improve troubleshooting workflows
- Build lightweight monitoring utilities
